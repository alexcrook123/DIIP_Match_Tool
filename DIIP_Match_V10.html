Intern → Host Company Matcher

This tool is designed to help align interns to host companies. Taking the information provided by the intern and entering it into the relevant boxes below will return a list of companies, ranked by alignment. This alignment is currently based only on Location, Stream, Area of Engineering Specialty.

With the score, there is a spreadsheet with each company arranged alphabetically from the state they are in. You can enter the intern’s name and score into this spreadsheet. The result should be a spreadsheet with a number of interns under each company.
